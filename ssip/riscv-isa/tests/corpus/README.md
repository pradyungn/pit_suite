# Instruction test corpus

This directory contains lists of instructions and their encodings for various RISC-V extensions.

The assembly was written by hand and the encodings were generated by either assembling, or using
this excellent online encoder/decoder in the case of branch/jump instructions:
<https://luplab.gitlab.io/rvcodecjs/>. Many thanks to the developers of that tool.
